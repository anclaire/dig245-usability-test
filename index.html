WIKIPEDIA

CHANGE LANGUAGE
Time: 43.83 seconds

User was confused by location of the “language” option and initially looked at the top bar, then at the bottom. A “change language” link at the top might be helpful.

User tried the “command P” function initially rather than looking at the left toolbar. The many options on the left toolbar was overwhelming.
FIND BIRTHDAY
Time: 17.15 seconds

User knew to use the search bar and found Abraham Lincoln’s birthday on his page quickly.

RANDOM ARTICLE
Time: 27.70 seconds

Because the user had developed better recall, it was easier to find the function at this point despite the cluttered left panel.
CITE PAGE
Time: 17.05 seconds

The user first looked at the reference section at the bottom of the article then looked in the left panel to find the cite option.


SUMMARY & SUGGESTIONS
User developed better recall throughout test but found site “overwhelming.” To improve the site, make the “Language” button more prominent, expand the “Tools” panel so that it’s less cramped, expand the search bar, and inclulde a “Cite” option in the resource section. These steps may improve user experiences when navigating Wikipedia.
