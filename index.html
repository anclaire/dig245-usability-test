RESULTS & SUGGESTIONS

User developed better recall throughout test but found site “overwhelming.” To improve the site, make the “Language” button more prominent, expand the “Tools” panel so that it’s less cramped, expand the search bar, and inclulde a “Cite” option in the resource section. These steps may improve user experiences when navigating Wikipedia.
